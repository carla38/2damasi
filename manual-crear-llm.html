Manual Crear LLM

February 18, 2026

El objetivo de esta tarea es crear nuestro propio modelo de lenguaje (LLM), entrenado a partir de una obra de Shakespeare. A continuación se redactará paso a paso el proceso seguido para conseguirlo.

Paso 1

Lo primero ha sido crear un directorio dedicado a esta tarea e iniciar un entorno virtual para la realización del proyecto. También habrá que instalar las librerías necesarias.

 

Paso 2

En este paso se realiza el procesamiento de datos y tokenización. Se crea el vocabulario que serán los caracteres que reconocerá la IA. Como la máquina solo reconoce números, primero habrá que convertir cada caracter a un número, y después se daría el caso inverso, necesario para que podamos ver la respuesta de la IA como letras y no números. Entonces todos nuestros datos (la obra) se convertirán en una lista de números (enconde), para posteriormente volverla a convertir en texto legible  (decode). Este sería el proceso de tokenización.

Se ejecuta data.py, que descargará el texto que se usará para entrenar, mostrará la longitud del dataset y los caracteres que conforman el vocabulario. Se confirma el guardado de los datos procesados en los archivos train.pt y val.pt, que se usarán para el proceso de entrenamiento, y meta.pkl, que es el diccionario que relacionará que número pertenece a qué caracter.

 

Paso 3

Se ejecuta el comando python model.py, que dará comienzo al entrenamiento del modelo. Este paso llevará un tiempo.

Esperando a que se completara este proceso descargamos la herramienta Mactop, diseñada para mostrar las métricas del dispositivo en tiempo real. Con ella pudimos comprobar que el dispositivo estaba usando un 100% de la GPU durante el entrenamiento.

 

Prueba Final

Finalizado el proceso de entrenamiento, cargamos el modelo con python chat.py. Como se puede observar en las capturas, primeros pasamos un texto, y nuestra IA nos responde en base a lo aprendido.
